Slendytubbies Android Demo

This project is to preserve this lost APK.

Slendytubbies Android Demo was an allegedly unfinished port of Slendytubbies 1.

We are working on porting this game back to PC as an archive.

 

As of May 7, 2023, only the main menu has been completed.
